# ML-Phishing-Detection-System

This repository contains an end-to-end, machine learning-driven security application designed to identify and classify phishing attempts. Built entirely in Python, the system covers the full data science lifecycle—from raw data processing and feature engineering to model training, evaluation, and live application deployment.

## File Structure
* **app.py** - The main application interface/API to run the detector.
* **train.py** - Script to train the machine learning model on the dataset.
* **detect.py** - Script for handling the prediction and detection logic.
* **features.py** - Feature extraction and engineering from the raw input data.
* **phish_model.pkl** - The serialized, trained machine learning model.
* **dataset_phishing.csv** - The dataset used for training and testing.
